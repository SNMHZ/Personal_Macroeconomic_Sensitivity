# 경제변수를 활용한 초개인화 서비스 제안

> 딥러닝을 통해 경제변수 변화에 따른 금융시장 참여자의 행동 변화를 예측·분석하여 초개인화된 금융상품과 서비스를 추천 및 개발할 수 있도록 한다

_리포지토리의 코드는 pytorch로 구현한 분류 모델입니다.<br> 세부 구조 및 정보는 아래 분석 결과 문서에서 확인할 수 있습니다._
